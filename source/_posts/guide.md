### Atlas

- **项目地址**：[Atlas 插件](https://gitee.com/Nwflower/atlas)
- **插件简介**：适用于V3版本及以上Yunzai-Bot的图鉴查询插件，可通过Yunzai-Bot查询游戏图鉴，插件代码严格遵循ES6规范
- **支持的图鉴库**：
  - 原神图鉴库：Genshin-Atlas
  - 崩坏：星穹铁道图鉴库：star-rail-atlas
  - 绝区零图鉴库：zzz-atlas
- **安装方法**：
  1. **下载插件本体**：
     - GitHub：
       ```bash
       git clone --depth=1 https://github.com/Nwflower/atlas ./plugins/Atlas/
       ```
     - Gitee镜像：
       ```bash
       git clone --depth=1 https://gitee.com/Nwflower/atlas ./plugins/Atlas/
       ```
  2. **获取图片资源**：
     - 原神：发送`#图鉴升级`
     - 星穹铁道：发送`#星铁图鉴升级`
     - 绝区零：发送`#绝区零图鉴升级`
  3. **插件更新**：发送`#图鉴插件升级`

### arknights-plugin

- **项目地址**：[arknights-plugin 插件](https://github.com/gxy12345/arknights-plugin)
- **插件简介**：基于森空岛API开发的Yunzai Bot明日方舟插件
- **安装方法**：
  1. **下载插件本体**：
     - GitHub：
       ```bash
       git clone https://github.com/gxy12345/arknights-plugin.git ./plugins/arknights-plugin/
       pnpm install --filter=arknights-plugin
       ```
     - Gitee镜像：
       ```bash
       git clone https://gitee.com/windoge/arknights-plugin.git ./plugins/arknights-plugin/
       pnpm install --filter=arknights-plugin
       ```
- **主要功能**：
  - **森空岛cred管理**：绑定、扫码绑定、删除、查询cred和token
  - **游戏信息查询**：便签、签到、抽卡记录、抽卡分析、干员练度卡片、练度统计、练度分析、养成计算、肉鸽、干员别名、理智、剿灭、日常/周常
  - **其他查询**：公招查询、刷图推荐、公告查询、活动查询、养成统计
  - **MAA远程控制**：配合MAA远控API服务使用

### earth-k-plugin

- **项目地址**：[earth-k-plugin 插件](https://github.com/SmallK111407/earth-k-plugin)
- **插件简介**：Earth-K-Plugin 是一个 Yunzai-Bot 的附属插件，主要提供了原史及部分功能

- **安装方法**：
  1. **下载插件本体**：
     - GitHub：
       ```bash
       git clone --depth=1 https://github.com/SmallK111407/earth-k-plugin.git ./plugins/earth-k-plugin/
       ```

       ```bash
       git clone --depth=1 https://gitee.com/SmallK111407/earth-k-plugin.git ./plugins/earth-k-plugin/
       ```

- **项目地址**：[yenai-plugin 插件](https://gitee.com/yeyang52/yenai-plugin)
- **插件简介**：Yenai-Plugin 是一个 Yunzai-Bot 的扩展插件，提供对 Bot 的一些便捷操作
- **注意事项**：
  - 该分支为 yenai-plugin v2 版本，配置文件进行重构，无法直接从 v1 进行迁移，可能需要重新安装以确保不会发生未知问题
  - 如果觉得项目有帮助，请给一个星星（Starred）
- **安装方法**：
  1. **下载插件本体**（两种方法二选一）：
     - GitHub：
       ```bash
       git clone -b v2 --depth=1 https://github.com/yeyang52/yenai-plugin.git ./plugins/yenai-plugin
       ```
     - Gitee：
       ```bash
       git clone -b v2 --depth=1 https://gitee.com/yeyang52/yenai-plugin.git ./plugins/yenai-plugin
       ```
  2. **安装依赖**：
     ```bash
     pnpm install
     ```
