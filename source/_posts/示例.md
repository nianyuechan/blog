## 主题特性

### 支持的功能

- ✨ 完整的博客功能
- 🔄 兼容 Hexo6 及以上版本
- 📱 响应式布局
- 🌙 暗黑模式支持
- 🅰️ i18n 支持
- 🖥️ 代码高亮与复制
- ➗ KaTeX / MathJax3 数学公式支持
- 📊 Mermaid 流程图支持
- 🔍 搜索集成
- 💬 多评论系统支持
- 🎵 音乐播放器支持（Aplayer + Meting）
- 🖼️ 图片懒加载
- ⚡ 加载动画
- 🖱️ 鼠标特效
- 👾 Live2D 集成

### 配置示例

```yaml
# 基本配置
title: 大家の幻想乡
author: Toshikoshi
language: zh-CN

# 主题配置
theme: reimu

# 部署配置
deploy:
  type: git
  repo: https://github.com/nianyuechan/blog
  branch: gh-pages

# 数学公式
markdown_it_plus:
  rawLaTeX: true
```


## 提示和技巧

1. **封面设置**：在文章的 Front-matter 中设置 `banner` 和 `cover` 来分别控制头图和卡片封面
2. **置顶文章**：添加 `sticky: true` 来置顶文章
3. **目录设置**：添加 `toc: true` 来显示文章目录
4. **代码高亮**：主题支持多种编程语言的代码高亮
5. **响应式设计**：网站在手机、平板和桌面设备上都能良好显示
6. **暗黑模式**：支持自动根据系统设置切换暗黑模式
